What's the deal with let, var, const?

Confused White Persian Guardian GIF from Confused GIFs

In JavaScript, there are three ways to declare variables:

  • var width = 100;
  • let height = 200;
  • const key = 'abc123';

Let is a little stricter than var. Let and var also have different scope. Let is block level scope (anything between the curly braces) whereas var is function level scope (available in the function that it is defined in).
